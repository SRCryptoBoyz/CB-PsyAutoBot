# CB-PsyAutoBot
BOT AUTO DAILY CHECK-IN


1. SET UP 

git clone [https://github.com/SRCryptoBoyz/CB-PsyAutoBot.git]

cd CB-PsyAutoBot



2. Install Dependencies =    npm install



3. Enter Your Bearer Token =   nano token.txt



4. If you want to use proxy , Enter Your Proxy = 
   nano proxy.txt



5. Run Bot = 
   npm start
